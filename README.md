# Barua's Sandwich Corner

Barua's Sandwich Corner is a static restaurant website created using HTML and CSS.  
The website includes multiple pages for a sandwich restaurant, including a homepage, menu, services, staff, testimonials, contact form, directions, and related links.

## Project Overview

This project was created to practice building a complete static website using only HTML and CSS.  
It focuses on page structure, navigation, styling, images, forms, tables, and responsive design.

## Pages Included

- Home page
- Menu page
- Services page
- Contact page
- About page
- Staff page
- Testimonials page
- Direction page
- Link page

## Features

- Multi-page website
- Navigation menu
- Hero section with background image
- Restaurant menu cards
- Contact form
- Service hours table
- Staff profile cards
- Customer testimonials
- Video section
- Footer with contact information
- Responsive layout for smaller screens

## Technologies Used

- HTML5
- CSS3
